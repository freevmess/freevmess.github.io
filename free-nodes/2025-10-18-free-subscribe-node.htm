三、高阶配置技巧

3.1 节点优选策略

通过url-test策略自动选择延迟最低的节点：
```yaml
proxies:
- name: "美国节点A"
type: ss
server: us1.example.com
port: 443
# ...其他节点...

proxy-groups:
- name: "自动优选"
type: url-test
proxies: ["美国节点A", "日本节点B"]
url: "http://www.gstatic.com/generate_204"
interval: 300
```

3.2 规则集动态更新

利用Rule Provider实现自动更新：
yaml rule-providers: anticensorship: type: http url: "https://cdn.jsdelivr.net/gh/Loyalsoldier/clash-rules@release/gfw.txt" interval: 86400 behavior: domain
